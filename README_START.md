# Akyl Cheshmesi mobile starter

1. Copy `.env.example` to `.env`
2. Put your app icons into `assets/images/`
3. Replace `REPLACE_WITH_EAS_PROJECT_ID` in `app.config.ts`
4. Run `npm install`
5. Run `npx expo start`

For web:
- `npm run web`

For EAS:
- `eas login`
- `eas build:configure`
- `eas build --platform android --profile preview`
